# Smartphone Specifications and Market Analysis

## Overview
This project analyzes smartphone specifications, ratings, and pricing data using statistical techniques to identify market trends and consumer preferences.

## Objectives
- Compare smartphone brands and specifications
- Analyze pricing patterns
- Identify relationships between ratings and prices
- Perform correlation and regression analysis

## Dataset
- Source: Kaggle Flipkart Mobiles Dataset
- Records: 2,647 smartphones

## Tools Used
- Microsoft Excel
- Minitab
- Statistical Analysis

## Analysis Performed
### Descriptive Statistics
- Mean
- Median
- Standard Deviation
- Quartiles

### Data Visualization
- Histograms
- Boxplots
- Pie Charts
- Bar Charts
- Scatter Plots

### Advanced Analysis
- Correlation Analysis
- Linear Regression
- Univariate Analysis
- Bivariate Analysis

## Key Findings
- Strong positive correlation between Original Price and Selling Price (r = 0.968).
- Smartphone ratings showed weak correlation with price.
- Premium brands such as Apple and Samsung occupied higher price segments.
- Budget brands focused on affordable devices.
